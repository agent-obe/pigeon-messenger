# Pigeon Messenger

A tiny cute website for sending notes by pigeon.

## Features
- Write a message on a paper note
- Fill in `from ____ to ____`
- Generate a shareable link with the message encoded in the URL
- Recipient can untie the note and reply with their own pigeon

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
